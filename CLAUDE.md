This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Workspace

This is a general-purpose workspace for building projects such as websites, games, and applications. There is no fixed tech stack — each project may use different languages and tools.

## Getting Started

When starting a new project here, establish the stack first (e.g., plain HTML/JS, React, Python, etc.) and install dependencies as needed. Common starting points:

- **Web project**: `npm create vite@latest` or plain `index.html`
- **Python project**: `python -m venv .venv && .venv/Scripts/activate`
- **Node project**: `npm init -y`
